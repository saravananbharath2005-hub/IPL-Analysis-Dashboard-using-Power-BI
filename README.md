🏏 IPL Data Analysis Dashboard

Overview

An interactive Power BI dashboard that analyzes IPL data from 2008–2024. The project provides insights into team performance, player statistics, match results, and venue trends through dynamic visualizations and data-driven analytics.

Features

Team performance analysis

Player statistics and rankings

Venue-wise match insights

Season-wise IPL trends

Interactive filters and dashboards


Tools Used

Power BI

DAX

Excel/CSV Datasets


Key Insights

Team win distributions and performance trends

Top-performing batsmen and bowlers

Venue impact on match outcomes

Seasonal IPL statistics and comparisons
